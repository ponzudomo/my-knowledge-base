- `change-summary/` 内に、`file-descriptions/` の中身をコピーし、上に書いた `file-descriptions/` の命名規則に従って `(新しく作ったファイル)` について説明するためのファイルを用意しなさい。
- `change-summary/` のセットアップが終わったら、各ファイルに該当する変更内容をまとめること。変更しなかったファイルについては、中身を空にせよ。
